================================================================================
                        SKILLSHARE APP - COMPLETE OVERVIEW
================================================================================

App Name    : SkillShare
Version     : 1.0.0+1
Platform    : Flutter (Android, iOS, Web, Windows, Linux, macOS)
Framework   : Flutter (Dart) with Firebase Backend
Description : A platform for skilled professionals to showcase their work,
              sell products, find jobs, and connect with customers and companies.

================================================================================
                              USER ROLES
================================================================================

The app supports 4 distinct user types:

1. CUSTOMER
   - Regular users who browse skilled professionals
   - Can chat with skilled users
   - Can purchase products from the shop
   - Can leave reviews and ratings
   - Can post service requests

2. SKILLED PERSON (skilled_person)
   - Service providers / freelancers
   - Can create a portfolio to showcase their work
   - Can list and sell products in the shop
   - Can apply for jobs posted by companies
   - Can receive service requests from customers
   - Must complete Aadhaar (ID) verification to go public

3. COMPANY
   - Organizations / businesses
   - Can post job listings
   - Can directly hire skilled professionals
   - Can chat with skilled persons for hiring

4. ADMIN
   - Platform administrator
   - Manages verifications, appeals, and platform moderation

================================================================================
                              APP MODULES / SCREENS
================================================================================

--- 1. AUTH MODULE (lib/screens/auth/) ---
   * login_screen.dart
     - Email & password login
     - Redirects to role-based home after login

   * signup_screen.dart
     - New user registration
     - Role selection (Customer / Skilled Person / Company)
     - Name, email, phone, and password fields

--- 2. SPLASH SCREEN (lib/screens/splash_screen.dart) ---
   - Displayed on app launch
   - Checks Firebase auth state
   - Routes to login or home based on session

--- 3. MAIN NAVIGATION (lib/screens/main_navigation.dart / main_screen.dart) ---
   - Bottom navigation bar
   - Tabs: Home, Shop, Jobs, Chat, Profile
   - Role-aware tab visibility

--- 4. HOME MODULE (lib/screens/home/) ---
   * home_screen.dart
     - Dashboard for all roles
     - Shows categories, featured skilled users
     - Quick access to services

   * explore_screen.dart
     - Browse and search skilled professionals
     - Filter by category, location, rating
     - Shows expert cards with skills and ratings

--- 5. SHOP MODULE (lib/screens/shop/) ---
   * shop_screen.dart
     - Browse all products listed by skilled users
     - Filter by category
     - Add to cart functionality

   * product_detail_screen.dart
     - Full product details (images, description, price)
     - Add to cart, buy now options
     - Seller info and contact

   * add_product_screen.dart
     - Skilled users list new products
     - Upload product images via Cloudinary
     - Set price, description, category

--- 6. JOBS MODULE (lib/screens/jobs/) ---
   * jobs_screen.dart
     - Browse open job listings
     - Filter by category and location
     - Companies and customers post jobs here

   * job_detail_screen.dart
     - Full job description
     - Apply for job (Skilled Person)
     - View applicants (Company/Customer)

   * create_job_screen.dart
     - Companies post new job listings
     - Set title, description, budget, category, location

--- 7. CHAT MODULE (lib/screens/chat/) ---
   * chats_screen.dart
     - List of all active conversations
     - Shows unread message count
     - Sorted by latest message

   * chat_detail_screen.dart
     - Real-time messaging with another user
     - Text messages
     - Work request integration (request/accept/reject work in chat)
     - Block/unblock user option

--- 8. PORTFOLIO MODULE (lib/screens/portfolio/) ---
   * portfolio_screen.dart
     - Display a skilled user's portfolio items
     - Viewable by all users

   * add_portfolio_item_screen.dart
     - Skilled users add portfolio entries
     - Upload images, set title, description, category

   * my_shop_screen.dart
     - Skilled user's own shop management
     - View, edit, delete own products

--- 9. PROFILE MODULE (lib/screens/profile/) ---
   * profile_screen.dart
     - View own profile
     - Edit profile info, profile photo
     - View portfolio and shop

   * profile_tab_screen.dart
     - Profile page as a bottom nav tab
     - Includes settings, logout

   * skilled_user_setup_screen.dart
     - Initial profile setup for Skilled Persons
     - Add skills, bio, location, Aadhaar for verification

   * customer_setup_screen.dart
     - Initial profile setup for Customers
     - Add name, location, preferences

   * company_setup_screen.dart
     - Initial profile setup for Companies
     - Add company name, description, location, industry

   * edit_skilled_profile_screen.dart
     - Edit existing skilled user profile
     - Update skills, bio, rates, visibility

================================================================================
                              DATA MODELS (lib/models/)
================================================================================

   user_model.dart            - Core user (uid, email, name, role, phone, photo)
   skilled_user_profile.dart  - Extended profile for skilled persons (skills,
                                bio, location, Aadhaar, verification status,
                                rating, portfolio, visibility)
   customer_profile.dart      - Extended profile for customers
   company_profile.dart       - Extended profile for companies (company name,
                                industry, location, job posts)
   product_model.dart         - Product listing (title, price, images, category,
                                sellerId, stock)
   cart_item_model.dart       - Shopping cart item (productId, qty, price)
   order_model.dart           - Order details (items, total, status, timestamps)
   job_model.dart             - Job posting (title, description, budget, status,
                                category, location, applicants)
   service_model.dart         - Service offered by a skilled user
   service_request_model.dart - Service request from customer to skilled person
   chat_model.dart            - Chat conversation and message data
   review_model.dart          - Rating and review (stars, comment, reviewer)
   portfolio_model.dart       - Portfolio item (title, images, description)
   appeal_model.dart          - Appeal filed by a user (for bans/rejections)

================================================================================
                              SERVICES (lib/services/)
================================================================================

   auth_service.dart
     - Firebase Authentication (Sign Up, Login, Logout)
     - Email/password auth with retry logic
     - User creation in Firestore on signup

   firestore_service.dart
     - Centralized Firestore database operations
     - CRUD for profiles, products, jobs, reviews, orders
     - Cart management (add, remove, clear)
     - Service requests (create, accept, reject)
     - Aadhaar verification and registry management
     - Block/unblock users
     - Appeals handling
     - Job applications

   chat_service.dart
     - Real-time chat using Firestore streams
     - Create/get chat rooms between users
     - Send and receive messages
     - Work request messages in chat
     - Block check before allowing messages

   cloudinary_service.dart
     - Upload images to Cloudinary CDN
     - Supports File upload (mobile) and Bytes upload (web)
     - Used for profile photos, portfolio, products, chat media

   portfolio_service.dart
     - Portfolio CRUD operations
     - Fetch portfolio for a skilled user

================================================================================
                          VERIFICATION SYSTEM
================================================================================

   Skilled users must go through a verification process:
   - Submit Aadhaar number (Indian national ID)
   - Aadhaar is checked for uniqueness in the aadhaar_registry collection
   - Verification statuses: pending -> approved / rejected
   - Only approved + public profiles appear in explore/search
   - Rejected users can file appeals
